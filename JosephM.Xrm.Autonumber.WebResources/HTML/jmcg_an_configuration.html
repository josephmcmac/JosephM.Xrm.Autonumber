Joseph Mcgregor Autonumber

This solution adds the capability to configure system generated unique autonumber fields

Autonumbers are configured by creating 'Autonumber' records

Note plugin triggers for the autonumber are automatically created by the system when autonumber records are created though note that the plugins are required to be turned on when the autonumbers records are created for this to work

Fields may be either integer or string type. For strings there is the option of having a prefix and/or seperator in the generated autonumber. There is also the option of having a parent autonumber where the number will be of the form AN-XXXXXX-YY where XXXXXX is the autonumber of the records 'parent' record which is referenced by a lookup field

See this article for more details https://josephmcmac.wordpress.com/2017/12/22/dynamics-365-autonumber-solution/
